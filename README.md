# Basic Unix Shell

A lightweight Unix shell written in C. This project was developed to understand the intricacies of process management, system calls, and shell architecture in a Unix-like environment.

## Features
- **Built-in Commands**: Supports basic commands like `cd`, `exit`, and `help`.
- **Process Execution**: Forks and executes programs using `execvp`.
- **Parsing**: Handles whitespace, quotes, and tokenization of input.
- **Error Handling**: Graceful error reporting for command failures.
